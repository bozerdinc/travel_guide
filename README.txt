Turkey Tourist Guide - Local Browser App

How to run:
1. Extract the ZIP file.
2. Open index.html with Chrome, Edge, or Firefox.
3. Use the destination cards, Start Tour, Listen, Previous/Next buttons.

Student voice recordings:
- Put MP3 files into assets/audio.
- Use these names: 01.mp3, 02.mp3, 03.mp3 ... 20.mp3.
- If an MP3 file is missing, the browser will read the English text with speech synthesis.

Content source:
- Images and English texts were prepared from the uploaded PowerPoint presentation.
